## 🚀 How to run the notebooks

```bash
# (recommended) create a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# install the core libraries
pip install jupyter numpy pandas matplotlib seaborn scikit-learn

# launch Jupyter
jupyter notebook
```
